## Task 2 – Client Lead Management System (Mini CRM)
## Mini CRM – Client Lead Management System
This project was developed as part of the Full Stack Web Development Track (FS).

## 📌 Task Requirement

- Build a simple CRM system to manage client leads generated from website contact forms.
- The system should allow:
- Storing client details
- Updating lead status
- Managing business workflow
- Performing CRUD operations

## 🏢 Project Overview

- Mini CRM is a web-based Client Lead Management System designed to manage customer leads efficiently.
- It allows businesses to:
- Add new customers
- View all customer records
- Update lead status (New / Old / Closed)
- Delete customers
- Automatically assign default status as "New"
- This system demonstrates backend integration, database management, and real-world CRUD functionality.

## 🛠 Tools & Technologies Used

## 🚀 How to Run the Project

1. Clone or download the repository.
2. Install dependencies:
## npm install
3. Make sure MongoDB is running locally.
4. Start the server:
## npm start
5. Open browser and visit:
## http://localhost:5000
